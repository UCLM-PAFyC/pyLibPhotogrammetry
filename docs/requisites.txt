Cuando se instalan librerias de python desde la consola de OSGeo4w se crean
en la carpeta correspondiente y pueden ser usadas por otras instalaciones de QGIS de la misma version:
C:\Users\dhernand\AppData\Roaming\Python\Python312\site-packages
Esto implica que si la hemos instalado antes para otra versión de QGIS no será necesario volver a instalarla
siempre que la versión de python coincida.

- OpenCV:
desde la consola de osgeo4w
python3.exe -m pip install --upgrade pip
pip install opencv-python
Si hubiera que desinstalarlo:
pip uninstall opencv-python

- Pint:
desde la consola de osgeo4w
pip install -U pint

- psutil
desde la consola de osgeo4w
pip install psutil

- xmltodict
desde la consola de osgeo4w
pip install xmltodict

- defusedxml -> NOOOOO
Para PIL exif
desde la consola de osgeo4w
pip install defusedxml

- exiftool
https://pypi.org/project/PyExifTool/
https://smarnach.github.io/pyexiftool/
desde la consola de osgeo4w
python -m pip install -U pyexiftool
Descargar exiftool para windows
Se ha instalado en: C:\Users\dhernand\AppData\Roaming\Python\Python312\site-packages\exiftool
